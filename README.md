# Ekohe interview assignment
A basic blog.

## Usage

```
# cd /path/to/ekohe_assignment
$ docker-compose up -d
$ bundle
$ bundle exec rails db:setup
$ bundle exec rails s
```

Open localhost:3000 in your browser.
